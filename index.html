Elder Voice Command Assistant

AI-Powered Voice Support System for Elderly Users

About the Project

This application enables elderly users to interact with computers using natural voice commands. Built using Deep Neural Networks, the system ensures accurate recognition and reliable responses with a strong focus on accessibility and emergency support.

Key Features

  • Voice-based command execution
  • Emergency assistance trigger
  • Text-to-speech system feedback
  • Simple, distraction-free interface
